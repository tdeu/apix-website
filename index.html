Start Building with APIX AI

Get up and running in less than 5 minutes

1

Install APIX AI

bash 
npm install -g apix-ai
# or
git clone https://github.com/your-repo/apix-ai
cd apix-ai && npm install
2

Set Up Environment

bash 
# Optional: For live blockchain operations
export HEDERA_ACCOUNT_ID="0.0.xxxxx"
export HEDERA_PRIVATE_KEY="302e020100..."
export OPENAI_API_KEY="sk-..."

Start Creating

bash 
# Interactive AI chat
apix chat

# Generate DeFi templates
apix add hts --name "LendingToken" --symbol "LEND"

# Create full application
apix generate fintech-lending
